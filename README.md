# Smart Traffic Signal Optimization System

## Overview

This project implements a Rule-Based Smart Traffic Signal Optimization System using Python, Tkinter, and OpenCV.

The system analyzes traffic scenarios, estimates traffic density, calculates priority scores, and optimizes traffic signal timings using IF–THEN decision rules.

## Features

* Traffic scenario selection using images
* OpenCV-based traffic density estimation
* Priority score calculation
* Rule-based AI decision engine
* Emergency vehicle priority override
* Traffic analysis dashboard
* Traffic report generation

## Technologies Used

* Python
* Tkinter
* OpenCV
* NumPy
* Pillow

## How to Run

1. Install dependencies:

```bash
pip install opencv-python numpy pillow
```

2. Run:

```bash
python traffic_gui.py
```
